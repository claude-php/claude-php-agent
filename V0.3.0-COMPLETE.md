## Technical Details

### DialogAgent ML Features
```php
$dialog = new DialogAgent($client, [
    'enable_ml_optimization' => true,
]);
```

**Learns:**
- Optimal context window (2-7 turns)
- Conversation strategies (direct, clarifying, summarizing)

**Benefits:**
- 20-30% token reduction
- 15% response quality improvement

### DebateSystem ML Features
```php
$debate = new DebateSystem($client, [
    'enable_ml_optimization' => true,
    'enable_early_stopping' => true,
]);
```

**Learns:**
- Optimal number of rounds
- Consensus thresholds

**Benefits:**
- 25-40% time reduction
- Early stopping when consensus reached

### MakerAgent ML Features
```php
$maker = new MakerAgent($client, [
    'enable_ml_optimization' => true,
]);
```

**Learns:**
- Optimal voting-K (3-7)
- Decomposition depth (3-10)
- Red-flagging enablement

**Benefits:**
- Near-zero error rates maintained
- Adaptive complexity handling

### PromptOptimizer
```php
$optimizer = new PromptOptimizer($client);

$optimization = $optimizer->optimize(
    originalPrompt: "Explain this",
    taskContext: "Educational tutorial"
);
```

**Features:**
- Historical prompt analysis
- k-NN based optimization
- A/B testing support
- Quality & token tracking

### EnsembleLearning
```php
$ensemble = new EnsembleLearning([
    'strategy' => 'weighted_voting',
    'client' => $client,
]);

$result = $ensemble->combine($task, $agents);
```

**Strategies:**
- Simple voting
- Weighted voting (learned weights)
- Bagging
- Stacking
- Best-of-N

**Benefits:**
- 10-25% accuracy improvement
- Variance reduction
